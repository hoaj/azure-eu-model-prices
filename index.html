Prices are zone-wide

Data Zone Standard token prices are billed at the EU-zone level — identical for Sweden Central and West Europe. The region toggle changes availability, not price.

Context tiers

gpt-5.4 / gpt-5.5 show the short-context rate. Long-context and pro tiers are billed higher — see the pricing page.

What's excluded

Cached-input, Batch and Provisioned rates are not shown. ada-002 has no Data Zone meter (price n/a). Audio / realtime / image / router models are out of scope.

Reasoning effort

Default reasoning_effort the model applies — what Cognigy's LLM Prompt Node uses (it has no reasoning control). Hover a model name for all options. — = not a reasoning model; n/d = default not documented by Microsoft.

Telco τ² benchmark

Agentic τ²-Bench Telecom score (% of tasks solved) from Artificial Analysis — higher is better. Uses AA's highest-effort variant, so the reasoning tier varies (gpt-5.4/5.5 at xhigh, gpt-5/5.1 at high); hover a score for the exact variant. — = not on the leaderboard (embeddings, gpt-4o-mini).

Cognigy support

Scraped from Cognigy's model-support page — Microsoft Azure OpenAI section only. Chat models show LLM Prompt Node support; embeddings show Knowledge Search support. — = not listed (the reasoning o-series).

Kept fresh

Regenerated daily by a GitHub Action that re-queries the Azure Retail Prices API (DKK + USD), re-checks region availability, and re-scrapes Cognigy support.
